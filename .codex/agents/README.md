These are repo-specific specialist profiles for LittleDownloader.

Pick the narrowest matching agent first:

- `telegram-flow.md`: webhook intake, Telegram types, handlers, commands, and user-facing bot behavior.
- `download-pipeline.md`: `DownloadJob`, `YtdlpDownloader`, quotas, storage layout, and notifications.
- `model-layer.md`: `User`, `Chat`, `Download`, model contracts, scopes, enums, defaults, and model-level specs.
- `data-ops.md`: migrations, rake tasks, reconciliation, and operational maintenance.
- `test-guardian.md`: specs, regression coverage, and CI safety.
- `reviewer.md`: code review, regression risks, and testing-gap analysis.

All specialists inherit the repo-wide guardrails from `AGENTS.md`.
