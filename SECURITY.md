# Security Policy

## Supported Versions

Only the `main` branch is currently supported.

Security fixes are applied to the latest version of the codebase.
Older commits or forks are not actively maintained.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately via one of the following channels:

- GitHub Security Advisories (preferred, if available)
- Email: **ulyanovda8@gmail.com**

Please include:
- A clear description of the issue
- Steps to reproduce (if possible)
- Potential impact

You can expect an initial response within **7 days**.

There is no formal bug bounty program for this project.
